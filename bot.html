Bruce Bot Cheat Sheet

Bruce is the bot that lives in our Campfire chat room. He can make your work day easier by deploying code to our production and development servers, keep track of what you are working on for the rest of the team, and even help you have a little fun. This page tells you how to use him.

  • Select from:
  • Work Tasks
  • Sounds
  • Emoji

Work Tasks

clone

Librarians with premission can clone github repositories to either production or development:

Bruce clone SERVER GITHUB_SSH_URL [PATH]
SERVER
– dev :: gvsulib.com
– prod :: labs.library.gvsu.edu
[PATH]
– The relative path to the directory you want to clone into. If you don't specify a directory, Bruce will clone into the web server root directory.

If I wanted to clone the repository at https://github.com/gvsulib/migrator-production onto the production server at http://labs.library.gvsu.edu/awesome/migrator-production, I would type:

Bruce clone prod git@github.com:gvsulib/migrator-production.git awesome

If I just wanted to clone the repo to http://labs.library.gvsu.edu/migrator-production, then I would say:

Bruce clone prod git@github.com:gvsulib/migrator-production.git

deploy

If you have permissions, Bruce can deploy code from github to either the production or development server:

Bruce deploy SERVER PATH [BRANCH]
SERVER
– dev :: gvsulib.com
– prod :: labs.library.gvsu.edu

If I wanted to deploy the project in labs/summon_usage on our development server, I would type:

Bruce deploy dev labs/summon_usage

If you do not specify a branch, Bruce will deploy the master branch.

i am working on

Tell Bruce what you are currently working on, so we don't have to have as many status meetings:

Bruce I am working on [Text]

You can just enter free text for [Text]. Since I'm working on the Bruce Bot Cheat Sheet, I could type:

Bruce I am working on the Bruce Bot Cheat Sheet

Bruce will store your most current status update.

What is everyone working on?

Have Bruce output everyone's current project, as set with 'I am working on.'

Bruce What is everyone working on?

Bruce also understands the parameter 'today,' if you only want updates that have happened on today's date.

Bruce What is everyone working on today?

i am at/in

Tell Bruce where you are, so we don't have to hunt you down"

Bruce I am in/at [Text]

You can just enter free text for [Text]. Since I'm in Kirkhof, I could type:

Bruce I am in Kirkhof

Bruce will store your most current location update.

Where is everyone?

Have Bruce output everyone's location, as set with 'I am at.'

Bruce Where is everyone?

Bruce also understands the parameter 'today,' if you only want updates that have happened on today's date.

Bruce Where is everyone today?
